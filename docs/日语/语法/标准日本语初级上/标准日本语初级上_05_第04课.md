## 【语法解释】

### 1.”あります”和“います”

>&emsp;&emsp;表示事物存在时，最常用的谓语时“あります”和“います”。“あります”用与花、草、桌子等不具有意志的事物。“います”用于具有意志的人、动物或昆虫。
>
><strong>名[场所]に　名[物/人]が　あります/います</strong>`相当于汉语的“~有~”`
>
>- <ruby>部屋<rp>(</rp><rt>へや</rt><rp>)</rp>に　<rp>(</rp><rt></rt><rp>)</rp>机<rp>(</rp><rt>つくえ</rt><rp>)</rp>　が　あります。</ruby>` 房间里有桌子。`
>- <ruby>ここに　<rp>(</rp><rt></rt><rp>)</rp>本<rp>(</rp><rt>ほん</rt><rp>)</rp>　が　あります。 </ruby>`这里有书。`
>- <ruby>庭<rp>(</rp><rt>にわ</rt><rp>)</rp>に　<rp>(</rp><rt></rt><rp>)</rp>何<rp>(</rp><rt>なに</rt><rp>)</rp>　が　ありますか。 </ruby>`院子里有什么？`
>- <ruby>部屋<rp>(</rp><rt>へや</rt><rp>)</rp>に　<rp>(</rp><rt></rt><rp>)</rp>猫<rp>(</rp><rt>ねこ</rt><rp>)</rp>　が　います。 </ruby>`房间里一只猫。`
>- <ruby>公園<rp>(</rp><rt>こうえん</rt><rp>)</rp>に　<rp>(</rp><rt></rt><rp>)</rp>子供<rp>(</rp><rt>こども</rt><rp>)</rp>　が　います。 </ruby>`公园里有孩子。`
>- <ruby>あそこに　だれ　が　いますか。 </ruby>`那里有谁？`

## 【表达及词语讲解】

### 1.「<ruby>上<rp>(</rp><rt>うえ</rt><rp>)</rp></ruby>」所表示的范围

>&emsp;&emsp;日语中的“上（うえ）”所表示的范围比汉语的“上”窄，只表示垂直上方的范围。汉语的“墙上”意思是墙壁的表面，而日语的“壁（かべ）の上”意思是墙壁上方的天棚，而不是墙壁的表面。墙壁的表面不用“壁の上に”而说“壁に”。
>
>- <ruby>壁<rp>(</rp><rt>カベ</rt><rp>)</rp>にスイッチが　あります。</ruby>`墙上有开关`
>
>
>
>



### 2.ええと

> &emsp;&emsp;「ええと」是被别人问及某事，思考该如何回答时说的话。
>
> - <ruby>小野<rp>(</rp><rt>おの</rt><rp>)</rp>さん、<rp>(</rp><rt></rt><rp>)</rp>会社<rp>(</rp><rt>かいしゃ</rt><rp>)</rp>はどこにありますか。</ruby>`小野，公司在哪儿啊。`
>
>   ————　<ruby>ええと、ここです。</ruby>`嗯（在地图上找，思考答案）…在这儿。`
>
>
>
> 



### 3.<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>家族<rp>(</rp><rt>かぞく</rt><rp>)</rp></ruby>/<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp></ruby>/<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>両親<rp>(</rp><rt>りょうしん</rt><rp>)</rp></ruby>　[礼貌语言④]

>&emsp;&emsp;「<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>家族<rp>(</rp><rt>かぞく</rt><rp>)</rp></ruby>`家人`」、「<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp></ruby>`兄弟姐妹`」、「<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>両親<rp>(</rp><rt>りょうしん</rt><rp>)</rp></ruby>`父母`」，这三个词是说到别人亲属时的礼貌说法。在会话中用「<ruby>ご<rp>(</rp><rt></rt><rp>)</rp>家族<rp>(</rp><rt>かぞく</rt><rp>)</rp></ruby>」来指对方家人，「<ruby><rp>(</rp><rt></rt><rp>)</rp>家族<rp>(</rp><rt>かぞく</rt><rp>)</rp></ruby>」来指自己家人。但不是所有的名词前面都可以加“ご”
>
>
>
>



### 4.<ruby><rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp></ruby>

>&emsp;&emsp;「<ruby><rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp></ruby>」指同一父母的人，也用于仅是同父或者同母的情况。不仅仅指男性兄弟之间，兄妹、姐弟、姐妹也称为「<ruby><rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp></ruby>」
>
>- <ruby>ご<rp>(</rp><rt></rt><rp>)</rp>兄弟<rp>(</rp><rt>きょうだい</rt><rp>)</rp>がいますか。</ruby>`你有兄弟姐妹吗？`
>
>- <ruby>ええ、<rp>(</rp><rt></rt><rp>)</rp>妹<rp>(</rp><rt>いもうと</rt><rp>)</rp>が　います。</ruby>`有，有妹妹。`
>
> 
>
>



### 5.<ruby>ＪＲ<rp>(</rp><rt>ジェーアール</rt><rp>)</rp></ruby>、<ruby>地下鉄<rp>(</rp><rt>ちかてつ</rt><rp>)</rp></ruby>

>&emsp;&emsp;「<ruby>ＪＲ<rp>(</rp><rt>ジェーアール</rt><rp>)</rp></ruby>」日本铁路，前身是「<ruby>国鉄<rp>(</rp><rt>こくてつ</rt><rp>)</rp></ruby>」，前身的全称「<ruby>日本国有鉄道<rp>(</rp><rt>にほんこくゆうてつどう</rt><rp>)</rp></ruby>」。

>&emsp;&emsp;「<ruby>地下鉄<rp>(</rp><rt>ちかてつ</rt><rp>)</rp></ruby>」是地铁。

### 6.<ruby>一人<rp>(</rp><rt>ひとり</rt><rp>)</rp>暮<rp>(</rp><rt>く</rt><rp>)</rp>らし</ruby>
