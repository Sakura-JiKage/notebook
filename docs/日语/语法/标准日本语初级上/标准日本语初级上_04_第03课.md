## 【语法解释】

### 1.「ここ/そこ/あそこ」＋は＋名词＋です

>&emsp;&emsp;指示场所时，用「ここ」、「そこ」、「あそこ」。所表示的位置关系与「これ」、「それ」、「あれ」相同。
>
>- <ruby>ここは　デパートです。</ruby>`这里是百货商店。`
>- <ruby>そこは　<rp>(</rp><rt></rt><rp>)</rp>図書館<rp>(</rp><rt>としょかん</rt><rp>)</rp>です。</ruby>`那里是图书馆。`
>- <ruby>あそこは　<rp>(</rp><rt></rt><rp>)</rp>入り口<rp>(</rp><rt>いりぐち</rt><rp>)</rp>です。</ruby>`那儿是入口。`
>
>
>
>

### 2.名词＋名词场所＋です

>&emsp;&emsp;表示“名词”存在于“名词场所”。
>
>- <ruby>食堂<rp>(</rp><rt>しょくどう</rt><rp>)</rp>はデパートの<rp>(</rp><rt></rt><rp>)</rp>7階<rp>(</rp><rt>ななかい</rt><rp>)</rp>です。</ruby>`食堂在百货商店的7层。`
>- <ruby>トイレは　ここです。</ruby>`厕所在这儿。`
>- <ruby>小野<rp>(</rp><rt>おの</rt><rp>)</rp>さんは<rp>(</rp><rt></rt><rp>)</rp>事務所<rp>(</rp><rt>じむしょ</rt><rp>)</rp>です。</ruby>`小野女士在事务所。`
>
>&emsp;&emsp;「小野さんは　事務所です。」的汉语译文是”小野女士在事务所”。不能译为“小野女士是事务所”。日语的「です」比汉语的“是”有更为广泛的含义。在这里表示小野女士在什么地方。
>
>&emsp;&emsp;「トイレは　ここです。」意思是“厕所在这儿”，是将「トイレ」作为话题来阐述。而「ここはトイレです」是将「ここ」作为话题来阐述，意思是“这里是厕所”。
>
>
>
>

### 3.名词＋は＋どこですか

>&emsp;&emsp;用于询问存在的场所。
>
>- <ruby>トイレは　どこですか。</ruby>`厕所在哪儿？`
>  ————　<ruby>あちらです。</ruby>`在那儿。`
>
>- <ruby>あなたの　かばんは　どこですか。</ruby>`你的包在哪儿？`
>  ————　<ruby>わたしの　かばんは　ここです。</ruby>`我的包在这儿。`
>
>
>
>

### 4.名词＋も＋名词＋です

>&emsp;&emsp;助词「も」基本相当于汉语的“也”。
>
>- <ruby>ここは　<rp>(</rp><rt></rt><rp>)</rp>ＪＣ企画<rp>(</rp><rt>ジェーシーきかく</rt><rp>)</rp>のビルです。</ruby>`这里是JC策划公司的大楼。`
>  <ruby>あそこも<rp>(</rp><rt></rt><rp>)</rp>ＪＣ企画<rp>(</rp><rt>ジェーシーきかく</rt><rp>)</rp>のビルです。</ruby>`那里也是JC策划公司的大楼。`
>
>- <ruby>李<rp>(</rp><rt>り</rt><rp>)</rp>さんは　<rp>(</rp><rt></rt><rp>)</rp>中国人<rp>(</rp><rt>ちゅうごくじん</rt><rp>)</rp>です。</ruby>`小李是中国人。`
>  <ruby>張<rp>(</rp><rt>ちょう</rt><rp>)</rp>さんも<rp>(</rp><rt></rt><rp>)</rp>中国人<rp>(</rp><rt>ちゅうごくじん</rt><rp>)</rp>です。</ruby>`小张也是中国人。`
>
>- <ruby>あなたも<rp>(</rp><rt></rt><rp>)</rp>中国人<rp>(</rp><rt>ちゅうごくじん</rt><rp>)</rp>ですか。</ruby>`你也是中国人吗？`
>
>
>
>

### 5.名词1＋は＋名词2＋ですか、名词3＋ですか

>&emsp;&emsp;答案有多种可能，而询问其中的一种时，可以重复使用谓语「～ですか」。因为这里询问的是哪一种，不能用「はい」或「いいえ」来回答。
>
>- <ruby>かばん　<rp>(</rp><rt></rt><rp>)</rp>売り場<rp>(</rp><rt>うりば</rt><rp>)</rp>は　<rp>(</rp><rt></rt><rp>)</rp>一階<rp>(</rp><rt>いっかい</rt><rp>)</rp>　ですか、<rp>(</rp><rt></rt><rp>)</rp>二階<rp>(</rp><rt>にかい</rt><rp>)</rp>ですか。</ruby>`卖包的柜台在1层还是2层？`
>
>- <ruby>今日<rp>(</rp><rt>きょう</rt><rp>)</rp>は　<rp>(</rp><rt></rt><rp>)</rp>水曜日<rp>(</rp><rt>すいようび</rt><rp>)</rp>ですか、<rp>(</rp><rt></rt><rp>)</rp>木曜日<rp>(</rp><rt>もくようび</rt><rp>)</rp>ですか。</ruby>`今天是星期三还是星期四。`
>
>- <ruby>林<rp>(</rp><rt>はやし</rt><rp>)</rp>さんは　<rp>(</rp><rt></rt><rp>)</rp>韓国人<rp>(</rp><rt>かんこくじん</rt><rp>)</rp>ですか、<rp>(</rp><rt></rt><rp>)</rp>日本人<rp>(</rp><rt>にほんじん</rt><rp>)</rp>ですか、<rp>(</rp><rt></rt><rp>)</rp>中国人<rp>(</rp><rt>ちゅうごくじん</rt><rp>)</rp>ですか。</ruby>`林先生是韩国人？日本人？还是中国人？`
>  ————　<ruby>日本人<rp>(</rp><rt>にほんじん</rt><rp>)</rp>です。</ruby>`是日本人。`
>
>
>
>

### 6.名词＋は＋いくらですか

>&emsp;&emsp;询问价钱时，用「いくら」。
>
>- <ruby>これは　いくらですか。</ruby>`这个多少钱？`
>- <ruby>その　<rp>(</rp><rt></rt><rp>)</rp>服<rp>(</rp><rt>ふく</rt><rp>)</rp>は　いくらですか。</ruby>`那件衣服多少钱？`
>
>
>
>

## 【表达及词语讲解】

### 1.<ruby>1階<rp>(</rp><rt>いちかい</rt><rp>)</rp></ruby>　「量词」

>&emsp;&emsp;「<ruby>1階<rp>(</rp><rt>いちかい</rt><rp>)</rp></ruby>」的「<ruby>階<rp>(</rp><rt>かい</rt><rp>)</rp></ruby>」是量词。日语的量词和前面的数词搭配使用时，有时候会有多种发音的情况。参照【词汇表（量词）】
>
>
>
>(1) **数字＋量词＋の＋名词**　「量词计数名词」
>
>&emsp;&emsp;如果一个量词不是特定的(例如人数、时间等) ，那么在数字和量词后面则需要跟着量词正在计数的特定名词。
>
>- <ruby>2<rp>(</rp><rt></rt><rp>)</rp>匹<rp>(</rp><rt>ひき</rt><rp>)</rp>の<rp>(</rp><rt></rt><rp>)</rp>犬<rp>(</rp><rt>いぬ</rt><rp>)</rp>がいる。</ruby>`有两只狗。`
>- <ruby>三<rp>(</rp><rt>さん</rt><rp>)</rp>つのリンゴを<rp>(</rp><rt></rt><rp>)</rp>買<rp>(</rp><rt>か</rt><rp>)</rp>いました。</ruby>`我买了三个苹果`
>
>
>
>(2) **名词＋数字＋量词**　「量词计数名词」
>
>&emsp;&emsp;当一个量词是特定的(例如人数、时间等) ，那么数字和计数器就可以在句子中自己充当主语/宾语。
>
>- <ruby>二人<rp>(</rp><rt>ふたり</rt><rp>)</rp>は<rp>(</rp><rt></rt><rp>)</rp>一緒<rp>(</rp><rt>いっしょ</rt><rp>)</rp>に<rp>(</rp><rt></rt><rp>)</rp>昼ご飯<rp>(</rp><rt>ひるごはん</rt><rp>)</rp>を<rp>(</rp><rt></rt><rp>)</rp>食<rp>(</rp><rt>た</rt><rp>)</rp>べた。</ruby>`两个人一起吃了午饭。`
>- <ruby>あと<rp>(</rp><rt></rt><rp>)</rp>十分<rp>(</rp><rt>じゅうぶん</rt><rp>)</rp>で<rp>(</rp><rt></rt><rp>)</rp>食<rp>(</rp><rt>た</rt><rp>)</rp>べるよ。</ruby>`（我）再过十分钟就吃。`
>
>
>
>(3) **数字 + 量词 + 动词**　「量词作为副词」
>
>- <ruby>ビールを<rp>(</rp><rt></rt><rp>)</rp>二<rp>(</rp><rt>に</rt><rp>)</rp>本<rp>(</rp><rt>ほん</rt><rp>)</rp>飲<rp>(</rp><rt>の</rt><rp>)</rp>んだ。</ruby>`”啤酒 + 宾语 + 两瓶 + 喝”、(我) 喝了两瓶啤酒。`
>- <ruby>猫<rp>(</rp><rt>ねこ</rt><rp>)</rp>を<rp>(</rp><rt></rt><rp>)</rp>三<rp>(</rp><rt>さん</rt><rp>)</rp>匹<rp>(</rp><rt>ひき</rt><rp>)</rp>飼<rp>(</rp><rt>か</rt><rp>)</rp>っている。</ruby>`”猫 + 三只 + 有”、(我) 有三只猫。`
>
>
>
>

### 2.省略　「谓语的省略」

>&emsp;&emsp;在会话中，进一步询问时，一般省略已说过一遍的谓语部分。被省略的部分一般包含疑问词。
>
>&emsp;&emsp;例如，首先用「これは　いくらですか`这个多少钱？`」询问，接着进一步询问「あれは　いくらですか`那个多少钱？`」时，由于后半部分的「いくらですか」与第一句的后半部分相同，因而常常省略，只说「あれは（升调）？」就可以了。
>
>&emsp;&emsp;应用课文中，「マンションの　隣（となり）は？`公寓的旁边呢？`」也是同样将后半部分省略了，句尾同样要读升调。
>
>
>

### 3.礼貌语言　「地点」

>(1) **こちら/そちら/あちら/どちら**
>
>&emsp;&emsp;「こちら/そちら/あちら/どちら」是「ここ/そこ/あそこ/どこ」的礼貌说法。「こちら/そちら/あちら/どちら」原本是表示方向的词语。
>
>- <ruby>受付<rp>(</rp><rt>うけつけ</rt><rp>)</rp>はどこですか。</ruby>`接待处在哪儿？（一般说法）`
>  ————　<ruby>あそこです。</ruby>`在那儿。`
>
>- <ruby>受付<rp>(</rp><rt>うけつけ</rt><rp>)</rp>はどちらですか？</ruby>`接待处在哪里？（礼貌说法）`
>  ————　<ruby>あちらです。</ruby>`在那里。`
>
>
>
>(2) **お<ruby>国<rp>(</rp><rt>くに</rt><rp>)</rp></ruby>はどちらですか**
>
>&emsp;&emsp;询问来自哪个国家。“お国”比“国”更礼貌一些。不是所有的名词前都可以加”お“
>
>
>
>(3) **<ruby>会社<rp>(</rp><rt>かいしゃ</rt><rp>)</rp></ruby>はどちらですか。**
>
>&emsp;&emsp;有两个含义，第一个意思是你的公司在哪里？第二个意思是你是哪个公司的。具体哪个意思要根据上下文判断。
>
>- <ruby>会社<rp>(</rp><rt>かいしゃ</rt><rp>)</rp>はどちらですか。</ruby>`你的公司在哪里？/你是哪个公司的？`
>————　<ruby>回答①：<rp>(</rp><rt></rt><rp>)</rp>上海<rp>(</rp><rt>しゃんはい</rt><rp>)</rp>です。</ruby>`在上海`
>————　<ruby>回答②：<rp>(</rp><rt></rt><rp>)</rp>ＪＣ企画<rp>(</rp><rt>ジェーシーきかく</rt><rp>)</rp>です。</ruby>`JC策划公司。`
>
>
>
>

### 4.缩略词

>&emsp;&emsp;"パソコン"由"パーソナルコンピュータ"（英语的personal computer)的缩略而来。日语中有将一个长词缩短的情况。这样形成的词叫缩略词。
>
>- <ruby>コンピニエンスストア→コンピニ　</ruby>`便利店`
>- <ruby>デジタルカメラ→デジカメ</ruby>`数码相机`
>
>
>
>

### 5. あのう　「搭话」

>&emsp;&emsp;用于向对方搭话或欲引起对方注意。另外，在说到难以启齿的事情、开始一个新话题以及向别人提出请求时，也可用“あのう”。
>
>- <ruby>甲：あのう、<rp>(</rp><rt></rt><rp>)</rp>課長<rp>(</rp><rt>かちょう</rt><rp>)</rp>。</ruby>`对不起，科长。`
>  <ruby>乙：<rp>(</rp><rt></rt><rp>)</rp>何<rp>(</rp><rt>なん</rt><rp>)</rp>ですか。</ruby>`什么？`
>
>
>
>

### 6. ですか　「确认」

>&emsp;&emsp;用于进一步确认对方所说的信息，句尾读升调。
>
>- <ruby>あのう、<rp>(</rp><rt></rt><rp>)</rp>東京<rp>(</rp><rt>とうきょう</rt><rp>)</rp>の<rp>(</rp><rt></rt><rp>)</rp>地図<rp>(</rp><rt>ちず</rt><rp>)</rp>は　どこですか。</ruby>`请问，东京地图在什么地方？`
>  ————　<ruby>地図<rp>(</rp><rt>ちず</rt><rp>)</rp>ですか。そちらです。</ruby>`地图？在那儿。`
>
>&emsp;&emsp;在对方所说的内容后加“か”。重复一遍，可表示理解了对方所说的内容。这时句尾读降调。
>
>- <ruby>それは　パソコンです。</ruby>`那是个人电脑。`
>  ————　<ruby>パソコンですか。</ruby>`（降调）。是个人电脑啊。`
>
>
>
>
