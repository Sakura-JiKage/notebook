### ✅ **常用符号 & 装饰**

| 符号                                | 含义           | 例子                    |                                       |
| :---------------------------------- | :------------- | :---------------------- | ------------------------------------- |
| ✅                                   | 正确           | **✅ 正解！**            | `&#9989;`                             |
| <font color="green">&#10004;</font> |                |                         | `<font color="green">&#10004;</font>` |
| ❌                                   | 错误           | **❌ 誤り！**            | `&#10060;`                            |
| ⭕                                   | 正确（日式）   | **⭕ 大丈夫です！**      |                                       |
| ❎                                   | 错误（另一种） | **❎ ダメです！**        | `&#10062;`                            |
| 🔼                                   | 向上/提高      | **コストが🔝（上がる）** |                                       |
| 🔽                                   | 向下/降低      | **価格が🔽（下がる）**   |                                       |
| ➡️                                   | 指向/下一步    | **次に➡️**               |                                       |
| 🔄                                   | 循环/重复      | **練習を🔄**             |                                       |
| **🔜**                               | **状态变化**   | **過去 🔜 現在 🔜 未来**  |                                       |
| 💡                                   | 提示/灵感      | **💡 ヒント！**          |                                       |
| ❗                                   | 重要注意       | **❗ 注意！**            |                                       |
| ❓                                   | 疑问           | **❓ どういう意味？**    |                                       |

△、▽、▲、▼：都可以用“さんかく”打出来。　　　

◆、◇：都可以用“ひし”打出来。

￥：可以用“えん”打出来。　　　＄：可以用“どる”打出来。

■、□：可以用“しかく”打出来。
